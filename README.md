# 📱 Google Play Store Apps Analysis

## 📊 Project Overview
This project analyzes data from the Google Play Store to uncover insights about app performance, user ratings, categories, installs, and pricing trends.  
The goal is to understand what makes an app successful and identify patterns in the app market.

---

## 🎯 Objectives
- Analyze app categories and their popularity
- Identify top-rated apps
- Study the relationship between installs, ratings, and reviews
- Compare free vs paid apps
- Discover trends in pricing and app size

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📁 Dataset
- Source: Google Play Store dataset (CSV format)
- Contains details like:
  - App Name
  - Category
  - Rating
  - Reviews
  - Installs
  - Type (Free/Paid)
  - Price
  - Size

---

## 🔍 Key Analysis Performed
- Data Cleaning and Preprocessing
- Handling missing values
- Converting data types
- Exploratory Data Analysis (EDA)
- Visualization of trends and patterns

---

## 📈 Insights
- Most apps belong to categories like **Family**, **Game**, and **Tools**
- Free apps dominate the Play Store
- Higher installs generally correlate with better ratings
- Paid apps are fewer but often have higher ratings
- Certain categories generate more user engagement

---

## 📊 Visualizations
- Bar charts for category distribution
- Scatter plots for installs vs ratings
- Heatmaps for correlation analysis
- Pie charts for app types
